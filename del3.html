Vad är JavaScript?

JavaScript

Vad är JavaScript?

Det är ett skriptspråk, men som till skillnad från HTML och CSS klassas som ett programmeringsspråk. Ett skriptspråk specialiserar sig på att utföra uppgifter inom redan befintliga miljöer. JavaScript är tillsammans med HTML och CSS en av webbens tre grundläggande byggstenar där HTML står för innehållet, CSS för designen och JavaScript för funktionaliteten.

Vad är ECMA?

JavaScript

Vad är ECMA?

Det är en ideel organisation som utvecklar standarder inom hårdvara, kommunikation och programmeringsspråk.

Vad är ECMAScript?

JavaScript

Vad är ECMAScript?

ECMAScript i folkmun kallat ES är en standardisering av JavaScript. När JavaScript uppdaterades 2015 (ES6) så bestämde sig ECMA för att börja med årliga uppdateringar istället för att med långa mellanrum släppa stora uppdateringar med extremt många nya funktioner för utvecklare att sätta sig in i. I skrivande stund är vi inne på ES11 vilket hade release 2020, och under 2021 kommer ES12. När man pratar om kommande versioner så säger man ESNext.

Vad är TypeScript?

JavaScript

Vad är TypeScript?

Det är ett "Superset" av JavaScript; kod skriven i JS är redan giltlig TypeScript-kod. Förutom sin vanliga kod kan man dessutom lägga till typ-annoteringar, samt deklarera variabler som exempelvis "string", "number" och så vidare.

JavaScript Web Development Frameworks

Vad är AngularJS / Angular samt skillnaden mellan dessa två?

JavaScript Web Development Frameworks

Vad är AngulasJS / Angular samt skillnaden mellan dessa två?

Angular är ett stort och populärt ramverk för att bygga små till stora webbapplikationer. Angular är TypeScript baserat medan AngularJS är JavaScript baserat.

Vad är Vue?

JavaScript Web Development Frameworks

Vad är Vue?

Det är ett JavaScript baserat ramverk för att skapa användargränssnitt och single-page applikationer.

Vad är Ember?

JavaScript Web Development Frameworks

Vad är Ember?

Ember.js är ett JavaScript ramverk och de anses främst vara ett ramverk för webben, men det är även möjligt att skapa desktop samt mobilapplikationer. Ember har stort fokus på att använda best practice för att utveckla skalbara applikationer.

Vad är node.js?

JavaScript Web Development Frameworks

Vad är node.js?

Node.js är en open source-plattform där man kör JavaScript på servernivå (client-side). Node.js använder sig av Chromes v8 engine (JavaScript motor) som översätter JavaScript syntax till maskinkod.

JavaScript Web Development Libraries

Vad är jQuery?

JavaScript Web Development Libraries

Vad är jQuery?

Det är ett av världen mest populära JavaScript bibliotek. Det är designat bland annat för att göra det enklare att anropa och modifiera element på hemsidor (divs, bilder, länkar, paragrafer med mera), hantera händelser (musklick etc), CSS-modifikationer samt skapa effekter och animationer.

Vad är React?

JavaScript Web Development Libraries

Vad är React?

React är ett effektivt och flexibelt JavaScript bibliotek som används för att bygga användergränssnitt.

Vad är w3.js?

JavaScript Web Development Libraries

Vad är w3.js?

Det är ett JavaScript bibliotek som är framtaget för att göra webbutveckling enklare.

Vad chart.js

JavaScript Web Development Libraries

Vad chart.js?

Det är ett JavaScript bibliotek som används för datavisualisering genom diverse diagram med hjälp av Canvas element.

Non-JavaScript Web Development Frameworks

Vad är flask / Django?

Non-JavaScript Web Development Frameworks

Vad är flask / Django?

Båda är ramverk för att utveckla webbaserade applikationer samt hemsidor med Phyton. Flask är extra lämpligt för att utveckla API:er och Django är ett fullstack ramverk för webbapplikationer utan API-stöd. Flask ses som ett microframework, om än ett kompetent sådant.

Vad är Blazor?

Non-JavaScript Web Development Frameworks

Vad är Blazor?

Det är den del av det populära webbutvecklingsverktyget ASP.NET och låter utvecklare skapa interaktiva web US´s med C# och HTML, CSS

Vad är Ruby on rails?

Non-JavaScript Web Development Frameworks

Vad är Ruby on rails?

Det är ett framework för att utveckla databas-stdjande webbapplikationer och använder sig av det objekt-orienterade skriptspråket Ruby. RoR hjälper utvecklaren att följa designmönstret MVC (Model-View-Controller).
