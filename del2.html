Framework

Vad är ett ramverk?

Ett ramverk är förberett stöd för utveckling av program, it-system eller webbsidor. Enkelt uttryckt en kombination av färdigskriven kod och regler för struktur för program. Exempel på ramverk är .NET

Vad är ett bibliotek?

Framework
vs.
Library

Vad är ett bibliotek?

Ett bibliotek är en samling metoder / funktioner för att till exempel upprätta hålla DRY (don´t repeat yourself) regeln. Används samma kod vid fler ställen i din kod så skapar man en metod av den koden för att kalla på metoden istället för att upprepa koden vid flertal ställen. En samling information som underlättar programutveck­ling. Ett bibliotek kan omfatta färdigskriven programkod, dokumentation, klasser och värden.

The World Wide Web

Vad är webben?

The World Wide Web

Vad är webben?

Det är ett informations system där dokument och andra resurser är identifierade och hämtas genom URL:s.

Vad är Client?

The World Wide Web

Vad är Client?

Program som lägger över arbetsuppgifter på ett annat program: servern.

Vad betyder client side?

The World Wide Web

Vad betyder client side?

Det är en handling som äger rum på användarens enhet (klienten).

Vad betyder server?

The World Wide Web

Vad betyder server?

Man kan beskriva en server som en mjukvara eller hårdvara som har uppgiften att lagra filer, system och information som andra enheter har tillgång till genom servern.

Vad betyder server-side?

The World Wide Web

Vad betyder server-side?

Det är en handling som äger rum på serversidan.

Vad är HTTP (Hypertext
Transfer Protocol)?

The World Wide Web

Vad är HTTP (Hypertext Transfer Protocol)?

Det är kommunikationen mellan klienter och webservrar och det sker genom HTTP requests och HTTP Responses. En användare skickar via webläsaren en http request (client side) som sedan besvaras genom en webserver (server-side).

Vad är HTTPS (Hypertext
Transfer Protocol Secure)?

The World Wide Web

Vad är HTTPS (Hypertext Transfer Protocol Secure)?

Det är en förlängning av http som är huvud protokollet som används vid överföring av data mellan webbläsare och hemsidor. HTTPS är krypterat för att säkerställa högre säkerhet vid överföring av data.

Vad är HTTP/1.1 och
HTTP/2 och vad är
skillnaden mellan dessa?

The World Wide Web

Vad är HTTP/1.1 och HTTP/2 och vad är skillnaden mellan dessa?

HTTP/1.1 stödjer request pipelining vilket låter en klient skicka flera förfrågningar samtidigt till webservern. HTTP/2 tillåter en webserver att svara med mer data än vad en klient har bett om.

Vad är UDP?

mellan dessa?

The World Wide Web

Vad är TCP/IP och UDP, vad är skillnaden mellan dessa?

TCP/IP Är en akitektur för datakommunikation över nätverk. TCP/IP består av fyra lager; Länk, Internet, Transport och Applikation. UDP Är ett överföringsprotokoll som ligger i TCP/IP lagret för transport. UDP är enklare än TCP, och har nästan inga funktioner för att rätta över­förings­fel.

Vad är DNS (Domain Name System)?

The World Wide Web

Vad är DNS (Domain Name System)?

DNS är en funktion som översätter domän¬namn till IP-adresser som datorer kan hantera.

Vad är URI (Uniform Resource Identifier)?

The World Wide Web

Vad är URI (Uniform Resource Identifier)?

URI är en rad tecken som hjälper till att identifiera en resurs på webben, antingen genom att använda platsen, namnet eller båda.

Vad är URL (Uniform Resource Locater)?

The World Wide Web

Vad är URL (Uniform Resource Locater)?

URL är en unik adress som visar vägen till en websida.

Vad är URN (Uniform Resource Name)?

The World Wide Web

Vad är URN (Uniform Resource Name)?

En URN är en unik identifikation som beskriver ett digitalt objekt/dokument eller resurs som är oföränderligt och globalt unik.

Vad är skillnaden mellan URI/URL/URN?

The World Wide Web

Vad är skillnaden mellan URI/URL/URN?

URL visar bara vägen.
URN beskriver objektet.
URI identifierar en resurs på nätet antingen genom en URL eller URN eller båda.

Vad är Polling?

The World Wide Web

Vad är Polling?

Polling kontrollerar genom en slags avfrågning om enheten är ansluten och fungerar eller kvaliten på överföringen, för att meddela att enheten kan börja sända eller kontrollera om de finns ny information.

Vad är Web Sockets?

The World Wide Web

Vad är Web Sockets?

Websockets är en teknik för snabb tvåvägskommunikation mellan webbläsare och webbserver. Kommunikationen sker med duplex, det vill säga att meddelanden kan skickas i båda riktningarna samtidigt.

Vad är XML (Extensible Markup Language)?

The World Wide Web

Vad är XML (Extensible Markup Language)?

XML är en specifikation för att specialdesigna så kallade märkspråk. Till skillnad från HTML så kan XML även tagga webbsidans innehåll utöver utseende.

Vad är JSON?

The World Wide Web

Vad är JSON?

Det är ett kompakt textbaserat format som används för att utbyta data. JSON är utformat för att enkelt fungera med JavaScript, men kan idag användas av mer eller mindre alla programspråk för att skapa och läsa dessa filer.

Vad är WebAssembly?

The World Wide Web

Vad är WebAssembly?

Det är en relativt ny teknologi skapad för att köra applikationer skrivna i andra språk än JavaScript direkt i webbläsaren. Tanken med detta är att det ska vara så nära "native" prestanda som bara möjligt och ha korta laddningstider.

Vad är FTP/FTPS/SFTP och vad är skillnaden mellan dessa?

The World Wide Web

Vad är FTP/FTPS/SFTP och vad är skillnaden mellan dessa?

File Transfer Protocol – vanligt protokoll för att hämta och skicka filer från en dator till en annan genom internet.
FTPS är en förlängning till FTP, där överföringarna kräver lösenord eller certifikat för att få tillgång till filerna.
SFTP - Använder sig av SSH-2 men skulle kunna använda sig av andra protokoll som exempelvis TLS.
Skillnaden mellan dessa är att de använder sig av olika nivåer av kryptering eller ingen alls som med FTP.

Vad är SMTP/POP3/IMAP och vad används dessa till?

The World Wide Web

Vad är SMTP/POP3/IMAP och vad används dessa till?

Simple Mail Transfer protocol – ett protokoll som ofta används för sändning av e‑post från avsändaren till internetoperatören, som sedan sänder det till den slutliga mot­ta­garen. Post Office Protocol, version 3 – ett protokoll för mottagning av e-post. POP3 sköter om mottagning och lagring av abon­nentens e‑post på internetoperatörens server tills abonnenten hämtar posten till sin dator. Internet Message Access Protocol – ett av de två vanliga protokollen för mottagning av e‑post. IMAP ser till att abonnentens inkommande e‑post tas emot och lagras på internetoperatörens server.

Vad är SSH?

The World Wide Web

Vad är SSH?

Secure SHell – ett protokoll för inloggning på distans med säker identifiering. SSH kan användas för fjärrstyrning av datorer.

HTML & CSS

Vad är HTML/XHTML/HTML5 och vad är skillnaden mellan dessa?

HTML

Vad är HTML/XHTML/HTML5 och vad är skillnaden mellan dessa?

HTML, Hypertext markup language är ett standardspråk för att skapa hemsidor, HTML beskriver webbsidans struktur i form av taggar. XHTML är en förlängning av HTML, XHTML använder sig av XML vilket möjliggör maskinläsbara beskrivningar av den information som finns på sidan. HTML5 är en omarbetning av HTML, HTML5 är bland annat utformad för att klara rörliga bilder och ljud utan tilläggsprogram som Flash.

Vad är HTML5 Canvas?

HTML

Vad är HTML5 Canvas?

Canvas är ett nytt element i HTML5, som tillhandahåller programmeringsgränssnitt för att generera och återge bilder, diagram och animeringar dynamiskt. Canvas är ett sätt att koda bilder på webbsidor i HTML5. Bildelement som ska publiceras på det sättet markeras med taggen canvas. Det är i grunden ett enkelt format, men det kan användas för rörliga bilder (ani­me­ringar), och kan med rätt programmering användas för mer avancerade effekter.

Vad är CSS/CSS3 och vad är skillnaden emellan dessa?

CSS

Vad är CSS/CSS3 och vad är skillnaden emellan dessa?

CSS står för Cascading Style Sheet, vilket betyder att det är en stilmall, även kallat stilmallsspråk. Detta räknas precis som HTML INTE som ett programmeringsspråk. CSS är däremot beroende av ett markup språk som HTML för att kunna påverka presentationen av innehållet.

Vad är ett CSS-ramverk?

CSS

Vad är ett CSS-ramverk?

Ett css framework kan ses som ett bibliotek som gör det enklare och mer standardiserad webdesign, de flesta framework innehåller olika moduler och verktyg där de flesta har någon form av grid, vilket underlättar för responsiv design, ikoner, fonter knappar och formulär. Exempel på ett css ramverk är Bootstrap.

Vad är SASS?

CSS

Vad är SASS?

Det är en förlängning av CSS, och sägs vara ett av de mest mogna förlägningar som finns idag. Det är fullt kompatibelt med css så man kan använda alla libraries som vanligt. SASS introducerar även variabler till css för att upprätthålla, DRY (don´t repeat yourself). SASS baserat på JavaScript.

Vad är LESS?

CSS

Vad är LESS?

LESS är ett ramverk precis som SASS, det är i princip samma sak som SASS men det är baserat på Ruby istället för JavaScript och saknar vissa funktioner SASS har som exempelvis loopar.
